Projects »

If you are a faculty/staff member you can submit a project request. Graduate students must have a PI to submit a project request. Once submitted you can add any additional users to your project and request a resource allocation. If you are an undergraduate student you do not have access to submitting a project request. You can, however, search for your PI's username and request access to a project they already have.

Allocations »

Resource allocations are required if you want to gain access to IU's resources. You must have a project to request an allocation. Be sure to review the list of available resources. Users must be added to an allocation in order to gain access to its resource. A project can have multiple allocations.
